Le menu Jeu
Le menu jeu permet au jouer de controler les differents status du jeu en cours:
  • Jeu->Nouveau (Ctrl+N)
Termine la partie en cours et en demarre une nouvelle.
  • Jeu->Meilleurs Scores (Ctrl+M)
Affiche les dix meilleurs joueurs.
  • Jeu->Conseil (Ctrl+C)
Montre une solution possible, mais te retire une vie.
  • Jeu->Quitter (Ctrl+Q)
Si le score est dans les dix meilleurs, le sauvegarde, et quitte le jeu


Le menu Option
  • Option->Pause(Ctrl+P)
Mets le jeu en pause en arretant le chrono, et en cachant le plateau de jeu.


le menu Aide
  • Aide->Documentation (Ctrl+D)
Ouvre dans une nouvelle fenêtre, le manuel du jeu Shisen-sho.
  • Aide->A-propos (Ctrl-A)
Donne les informations sur le jeu et les auteurs.
